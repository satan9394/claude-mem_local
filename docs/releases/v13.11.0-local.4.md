# Claude-Mem Local v13.11.0-local.4

## Windows one-click suite support

This release adds the maintained, non-interactive CC Switch installation path used by the CC Switch MEM Suite. It configures loopback-only discovery, real-time session-model following, no fixed model, and no silent provider fallback.

Claude Code's ordinary model name is unchanged. Only Claude-Mem background usage is labelled `MEM` by the maintained CC Switch fork.

## Security boundary

- Memory, observations, transcripts, and the SQLite database remain local.
- Sanitized Claude-Mem summaries leave the machine only through the provider currently selected in CC Switch.
- The installer does not collect prompts, memory data, API keys, or provider secrets.
- All downloadable assets have SHA-256 checksums.

This is an independent fork release from `satan9394/claude-mem_local`, not an official upstream Claude-Mem package.
